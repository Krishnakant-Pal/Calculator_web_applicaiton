Welcome to Simple Flask Application

Introduction

This web Application is simple Flask-based calculator that allows you to perform basic arithmetic calculations.

Getting Started

  1. Enter the first number in the input field.
  2. Select the desired operation from the dropdown menu (addition, subtraction, multiplication, or division).
  3. Enter the second number in the next input field.
  4. Click on the "Calculate" button.

Features

  • Addition: The addition functionality allows you to add two numbers together. Simply enter the first number, select the addition operation, enter the second number, and click "Calculate."
  • Subtraction: With the subtraction feature, you can subtract one number from another. Enter the first number, select the subtraction operation, enter the second number, and click "Calculate."
  • Multiplication: For multiplication, enter the first number, select the multiplication operation, enter the second number, and click "Calculate" to obtain the result.
  • Division: Finally, the division feature enables you to divide one number by another. Enter the first number, select the division operation, enter the second number (non-zero), and click "Calculate" to see the quotient.

Error Handling

To ensure accurate calculations, our calculator includes error handling for common issues:

  • Division by zero: If you attempt to divide a number by zero, the calculator will display an error message, prompting you to enter a valid non-zero value for the second number.
  • Invalid input: The calculator will check for valid numerical input in the number fields and display an error message if non-numeric characters are entered.
